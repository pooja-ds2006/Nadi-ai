# 🌿 Nadi - AI Ayurvedic Digital Twin

Nadi is an AI-powered Ayurvedic Digital Twin web application that helps users understand their health through personalized Ayurvedic analysis. The application uses Google's Gemini AI to provide Dosha predictions, health insights, and AI-powered assistance.

## ✨ Features

- 🔐 User Login & Registration
- 🩺 Health Profile Management
- 🌿 Ayurvedic Dosha Quiz
- 🤖 AI Dosha Prediction (Google Gemini)
- 📊 Digital Twin Dashboard
- 💬 AI Health Chatbot
- ⏰ Health Reminders
- 📈 Progress Tracking
- ⭐ User Feedback System
- 👨‍💼 Admin Dashboard

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Nadi-ai.git
```

Go to the project folder

```bash
cd Nadi-ai
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run the application

```bash
npm run dev
```

Open your browser

```
http://localhost:3000
```

or

```
http://localhost:5173
```
